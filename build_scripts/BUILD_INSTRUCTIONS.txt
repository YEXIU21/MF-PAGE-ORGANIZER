# MF Page Organizer - Build Instructions

## Optimized Build System (Choose One):

### 1. **Folder Build** (Recommended for most users):
```bash
python build_exe_folder.py
```
- ✅ Fast startup (~2-3 seconds)
- ✅ Full PaddleOCR with models bundled
- ✅ PREVENTS bloat by excluding unnecessary packages (~300-500MB)
- ✅ Professional distribution
- 📁 Output: `dist/PageAutomation/PageAutomation.exe` + files

### 2. **One-File Build** (Optimized - Maximum portability):
```bash
python build_exe_onefile.py
```
- ✅ Single portable file (~500MB-1GB, NOT 5.60GB)
- ✅ Full PaddleOCR with models bundled
- ✅ PREventS bloat by excluding unnecessary packages

## Recommendations:

- 🏠 **Development/Testing**: Use Folder Build (fastest startup)
- 💼 **Professional Distribution**: Use Folder Build (best performance)  
- 📧 **Sharing/Email**: Use One-File Build (single file)
- 🌐 **Portable/USB**: Use One-File Build (maximum portability)

## Prerequisites:

Before building, ensure PaddleOCR models are downloaded:
```bash
python build_scripts/download_paddleocr_models.py
```

## Support Files:

- `download_paddleocr_models.py` - Downloads PaddleOCR models
- `hook-paddleocr.py` - PyInstaller hook for PaddleOCR
- `icon.ico` - Application icon (auto-generated)
