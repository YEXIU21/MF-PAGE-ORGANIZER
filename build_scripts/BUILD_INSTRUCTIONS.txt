# MF Page Organizer - Build Instructions

## Quick Start (Choose One):

1. **Fast Startup Build** (No splash, fastest launch):
   ```
   python build_exe.py
   ```

2. **Enhanced Build** (PyInstaller image splash):
   ```
   python build_exe_with_splash.py
   ```

3. **Custom Splash Build** (Tkinter progress bar):
   ```
   python build_exe_with_custom_splash.py
   ```

4. **★ One-File Build** (Single portable EXE):
   ```
   python build_exe_onefile.py
   ```

5. **★★ One-File + Models** (Fully offline EXE):
   ```
   python build_exe_onefile_with_models.py
   ```

## Build Features:

- **Fast Build**: Fastest startup, no splash screen, folder output
- **Enhanced Build**: Beautiful image splash, fully featured, folder output
- **Custom Build**: Animated progress bar, loading messages, folder output
- **★ One-File Build**: Single EXE file, fully portable, larger size (~200-500MB)
- **★★ One-File + Models**: Single EXE with models embedded, 100% offline (~800MB-1GB)

## PaddleOCR Support:

All builds now include:
- ✅ Full PaddleOCR bundling for offline OCR
- ✅ Roman numeral detection (vi, vii, viii, ix, x, xi, xii)
- ✅ Multi-position page number scanning
- ✅ Automatic model pre-download

## Output:

- **Folder builds**: `dist/[BuildName]/[BuildName].exe` + supporting files
- **One-file build**: `dist/PageAutomationOneFile.exe` (single file)
